# Backend Utility Scripts

This folder contains utility scripts for database and image management.

## Scripts

- `check_all_images.py` - Check all images in the database
- `check_api.py` - Test API endpoints
- `check_db.py` - Check database integrity
- `debug_db.py` - Debug database issues
- `fix_image.py` - Fix image-related issues
- `fix_slow_images.py` - Optimize slow-loading images
- `fix_ysl.py` - Fix YSL-specific issues
- `update_broken_images.py` - Update broken image references
- `update_db.py` - Database update utilities
- `update_user_name.py` - Update user names in database

## Usage

Run scripts from the project root:

```bash
python backend/scripts/script_name.py
```

Make sure your virtual environment is activated and the backend is configured properly.
